Aim:

To calculate phase difference between two sinusoidal signals using lissajous figure method.

Theory

  • When the amplitudes and frequencies of the two sinusoids are same and they only differ by a phase shift then their superimposed signal forms an ellipse. Different phase shifts form ellipses with different eccentricities and inclinations with the horizontal axis.

    • Procedure

  • 1. Click on Turn on to turn on the oscilloscope and feed it with sinusiodal inputs with parameters specified by the sliders.
  • 2. Set amplitude of first sinusoid to a desired value using the slider.
  • 3. Set amplitude of second sinusoid equal to that of the first sinusoid.
  • 4. Click on “Assign Random Phase Difference” button (this button whenever clicked assigns a random value of phase difference between the sinusoids being superimposed).
  • 5. Set frequency of first sinusoid to a desired value using the slider.
  • 6. Set frequency of second sinusoid equal to that of the first sinusoid.
  • 7. Calculate the value of phase difference using the formula given above and enter in the text box.
  • 8. Check your answer by clicking on “Check” button.
  • 9. To add readings in the table, click on “Enter readings” button.
  • 10. Above procedure will complete one set of readings repeat the same for more.
