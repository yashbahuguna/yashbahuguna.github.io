Aim:

To find the frequency of a sinusoidal signal using lissajous figure method.

Theory

When the frequencies of two sinusoids being superimposed is different arbitrary curves are formed we can calculate the ratio of their frequencies by the given formula.
Horizontal and vertical tangencies can be calculated by counting intersection with the x and y axes respectively.
If frequency of one signal is known then that of other can be calculated.

Example:
fy/fx = 4/2 = 2

Procedure

  • 1. Click on Turn on to turn on the oscilloscope and feed it with sinusiodal inputs with parameters specified by the sliders.
  • 2. Set amplitude of first sinusoid to a desired value using the slider.
  • 3. Set amplitude of second sinusoid to a desired value using the slider.
  • 4. Set frequency of first sinusoid to a desired value using the slider this will be reference or known frequency.
  • 5. Click on “Assign Random Frequency” button (this button whenever clicked assigns a random value of frequency to the second sinusoud).
  • 6. Calculate the ratio of two given frequencies using the formula above.
  • 7. Calculate the unknown frequency since frequency of the first sinusoid that is along the x axis is known.
  • 8. Check your answer by clicking on “Check” button.
  • 9. To add readings in the table, click on “Enter readings” button.
  • 10. Above procedure will complete one set of readings repeat the same for more.
